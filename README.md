# InsureVisionAI

InsureVisionAI is a Snap Spectacles-based XR roof inspection companion that helps insurance assessors carry out faster, more consistent, hands-free damage assessments in the field.

The experience is being built in **Lens Studio 5.15.4** for **Snap Spectacles**.

## Project Overview

InsureVisionAI guides assessors through a roof inspection workflow by helping them:

- inspect damage hands-free
- highlight likely damage areas
- track inspection progress
- answer guided assessment questions
- generate a faster claim-ready summary on site

The goal is to replace the usual stop-start process of clipboards, phone photos, and post-visit reporting with a more guided and consistent inspection workflow.

## Built With

- **Lens Studio 5.15.4**
- **Snap Spectacles**

## Getting Started

### Prerequisites

Download and install **Lens Studio 5.15.4**:

[https://ar.snap.com/download/v5-15-4](https://ar.snap.com/download/v5-15-4)

### Open the Project

1. Install Lens Studio 5.15.4
2. Clone this repository
3. Open the project in Lens Studio
4. Connect or deploy to Snap Spectacles for testing
